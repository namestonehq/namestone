# Get Domain

This GET route fetches the information for a domain

## Parameters

- `domain` (string)(Optional): The domain for the user name. If not included route will return all domains tied to your API key.

## Example

```
GET https://namestone.xyz/api/public_v1/get-domain?domain=example.xyz...
headers:
{
  `Authorization: YOUR_API_KEY`
}
```

## Curl Example

```
curl -X GET \
     -H 'Authorization: YOUR_API_KEY' \
     'https://namestone.xyz/api/public_v1/get-domain?domain=testbrand.eth'

```
