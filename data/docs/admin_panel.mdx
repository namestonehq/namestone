# Admin Panel

NameStone provides a no-code [admin panel](https://namestone.com/admin) for easy name management. Use this tool to delete, add, or edit names and records for enabled domains on Mainnet and Sepolia.

Settings are also provided in the admin panel to view or change the following:

- Make names public or private - allow your names to be discovered by projects that interact with our API.
- View API key
- Manage domain admins
